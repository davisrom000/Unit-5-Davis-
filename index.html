Text Formatting

There are three types of text formatting. Those are: bold, italics, and underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and headers elements are block elements. An inline element keeps its content on the same line as the content before it and after it. It does NOT create a new line. The bold, italics and underline elements are inline elements.

An image element is a block element which inserts pictures into the document. An image element is a self-closing element, which means it does not have an end tag. It has one attribute, src, which is a filepath that points to the image's locationn. Those filepaths can be either absolute or relative.

Simple Image Example

Image Attributes

There are 4 attributes commonly used for images. Width and height are used to resize the images. Alt allows to use text-description of images for accessibility and broken images. Title allows a hover-over text to appear.

Width and Height

The width and height can be measured in many different ways, but the more common ones are pixels and percentages. Using pixels, you set the specific precise size based on pixels. Using percentages, you set the dimensions based ona percentage of the entire page. Percentage based resizing will often respect aspect ratios while using pixels may not.

Alt Attribute

Every picture should at least have the alt attribute. The alt attribute helps with screen readers for visually impaired users and in the cases that the image is broken.

Relative vs. Absolute

A absolute filepath includes all the navigate to get to the server, folders, and files. The examples above are all absolute filepaths. An absolute filepath is used whe the image is stored on a different server or device than the HTML document.

A relative filepath only include the folder and file. The examples below are all relative filepaths. A relative filepath is used when the images are stored alongside the HTML document.

Links

The anchor element is an inline element and is way to include hypertext links, otherwise known as just links, in your document. An anchor is a reference point for other documents or files. It commonly has 3 attributes: href, target, and download.

Simple Links

LINK

In order for a link to work is MUST include the href attribute. This attribute is the filepath, either absolute or relative, to the document, website or file you wish to link.

Links Attributes

The target attribute allows you to decide where the link opens. Ther are 4 options, but only one is commonly used. _blank opens the linked document in a new window or tab. _self opens the link in the current window or tab. This is the default behavior. The two others are only used if you have a frame or other window INSIDE a window. We will not be discussing these.
