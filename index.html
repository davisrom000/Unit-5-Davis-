Text Formatting

There are three types of text formatting. Those are: bold, italics, and underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and headers elements are block elements. An inline element keeps its content on the same line as the content before it and after it. It does NOT create a new line. The bold, italics and underline elements are inline elements.

Vertical Spacing

There are two elements that help with vertical spacing between block elements. A line break moves the content down to a new line. Its similar to the ENTER key in Word/Docs. A horizontal rule is a line break which includes a horizontal bar across the page. Both are block elements.
